## Installation

Install my project with yarn

```bash
IOS:
 yarn
 cd ios 
 pod install
 cd ..
 react-native run-ios

Android:
 yarn
 react-native run-android
```
    
    
## Running Tests

To run tests, run the following command

```bash
  yarn test
```
